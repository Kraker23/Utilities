# UtilitiesNet.Funciones

## Descripción
UtilitiesNet.Funciones es una biblioteca de utilidades para .NET 9 que proporciona diversas funciones útiles para aplicaciones Windows Forms.

## Características
- Target Framework: .NET 9.0
- Soporte para Windows Forms
- Uso de referencias implícitas y habilitación de tipos anulables

## Uso
Para utilizar las funciones proporcionadas por esta biblioteca, simplemente agregue una referencia al paquete en su proyecto y utilice las funciones según sea necesario.
