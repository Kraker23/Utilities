# UtilitiesNet.Extensiones

## Descripción
UtilitiesNet.Extensiones es una biblioteca de utilidades para .NET 9 que incluye extensiones útiles para trabajar con Windows Forms y otras funcionalidades comunes.

## Características
- Soporte para .NET 9.0
- Extensiones para Windows Forms
- Utilidades comunes para el desarrollo de aplicaciones .NET
