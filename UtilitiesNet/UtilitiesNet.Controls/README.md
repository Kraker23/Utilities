# UtilitiesNet.Controls

## Descripción
UtilitiesNet.Controls es una biblioteca de controles personalizados para aplicaciones Windows Forms desarrollada en .NET 9.0.

## Características
- Controles personalizados para mejorar la interfaz de usuario.
- Integración con System.Data.SqlClient para acceso a bases de datos.
- Soporte para Windows Forms.

## Uso
Después de instalar el paquete, puede agregar los controles personalizados a su formulario de Windows Forms y configurarlos según sus necesidades.
