# UtilitiesNet.Objetos

## Descripción
UtilitiesNet.Objetos es un paquete .NET 9 que proporciona utilidades y objetos comunes para aplicaciones Windows Forms. Este paquete incluye referencias a Microsoft Office Interop para Outlook y System.Drawing.Common.

## Características
- Target Framework: .NET 9.0
- Soporte para Windows Forms
- Utiliza Microsoft Office Interop para Outlook
- Utiliza System.Drawing.Common para operaciones gráficas

## Uso
Para utilizar las funcionalidades de este paquete, simplemente agregue una referencia a `UtilitiesNet.Objetos` en su proyecto y comience a utilizar las clases y métodos proporcionados.

## Requisitos
- .NET 9.0
- Windows Forms
